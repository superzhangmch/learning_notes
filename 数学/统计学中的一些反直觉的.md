高维空间球, 大部分体积在球面上; 特别的, 一个dim-n 正方体, 内切一个dim-n 球, 则几乎全部体积在球外. 

### 詹姆斯–斯坦估计（James–Stein Estimator）

统计学里一个非常反直觉但很重要的结果: 当你同时估计 3 个或以上的参数时，把每个参数“往整体平均值拉一拉”，反而能整体更准。

### 随机游走中的反正弦定律（Arcsine Law）

在一个对称随机游走或 布朗运动 中: “待在正数一侧的时间比例”, 并不是集中在 50%，而是最可能接近 0 或 1。

也就是说：要么几乎一直在正的那边，要么几乎一直在负的那边，反而“正好一半时间在正的那边”最不可能。

对应炒股: 在很长一段交易历史里，你要么几乎一直处于盈利状态，要么几乎一直处于亏损状态，而“盈利和亏损时间各占一半”反而最少见。

反正弦定律说的是什么？定义一个量： 

$$
T_n = \frac{\text{在前 } n \text{ 步中，位置 > 0 的步数}}{n}
$$

这是一个 **比例**，介于 0 和 1 之间。 你以为的分布（直觉）: 大多数时候在 0.5 附近吧？”


```
概率
  ^
  |        /\
  |       /  \
  |______/____\____> T
        0.5
```

真正的分布（反正弦定律）, 形状是：

```
概率
  ^
  | \            /
  |  \          /
  |   \        /
  |____\______/____> T
      0      1
```

**在 0 和 1 附近概率最大！**
