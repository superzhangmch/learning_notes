数学上，对一个二维函数, 若知道穿过它的所有直线的线积分（即其 Radon 变换），则可以通过反演恢复原函数。注意: 不只是过中心的线, 而是每个方向的众多平行线都要算.

CT 的原理正基于此。某一平面上，不同方向、不同位置的 X 光穿过人体后，接收到的强度会不同（因为人体不同软硬、疏密区域对 X 光的阻拦程度不同；这一过程满足 Beer–Lambert 定律，对接收到的强度取对数后，可转化为沿射线路径的积分）。这样就能得到大量直线积分信息，进而反解出人体的剖面图像。注: MRI 核磁共振基于 Fourier 原理.

对一张灰度后的img, 不同方向与位置采样(可以这样: 完全随机的散乱直线, 每条线两个端点都独立地随机抽自图像边界), 并累加灰度值(累加值即积分), 于是得到一系列的 `{线: 累加值}`, 从它就能反演恢复原img.

CT 重建 ≈ 解一个大型稀疏线性方程组。 

1. 把图像看成未知数。 一张 128×128 的图 = 16384 个未知像素值，记成一个长向量 x。

2. 每条"射线" = 一个线性方程。 在图上画一条直线（或曲线），把它穿过的所有像素值加起来，得到一个数 b_i。这个数就是 "这些像素的某种线性组合"，对应一个方程：a_i · x = b_i，其中 a_i 是个稀疏向量（穿到的像素是 1，没穿到的是 0）。

3. 很多条射线 = 很多方程。 把所有方程堆起来就是矩阵形式 A x = b。A 是稀疏矩阵（M 行 × N 列，M = 射线数，N = 像素数）。

4. 求解。 用 ART（Kaczmarz 迭代）：每次拿一行 a_i，把当前估计 x 投影到 "a_i · x = b_i" 这个超平面上，循环过完所有方程，反复迭代直到收敛。

5. 这就是全部数学。 不管几何怎么变（直线、贝塞尔曲线、随机走），只要每条路径还是"沿路径求和"，就是线性方程；只是 A 的结构变了，求解器照样能跑。
