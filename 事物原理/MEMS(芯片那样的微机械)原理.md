MEMS 类的微机械结构(投影仪的 DLP DMD 微镜, 手机上的加速度传感器, 麦克风等等), 其实是用类似造cpu那样的方法造出来的- 光学蚀刻, 一层层雕刻出来的. 只是 cpu 是雕出了静态的, 而这类结构是雕出了动态的. 

特别对于加速度传感器, 乃在硅(Si)上雕去多余部分, 形成了 Si 做的弹簧与质量快(不过时梳子形的), 从而可以感受力与加速度. 然后通过这个位移, 探测到电容变化, 从而探测出加速度.

另外, 现在的晶振, 其实内部真的是个物理小音叉, 只是靠电来驱动振动.

# MEMS 微机械结构

### 制造工艺

MEMS 用的正是**半导体微加工工艺**：
- 光刻（Photolithography）
- 深反应离子刻蚀（DRIE, Deep Reactive Ion Etching）—— MEMS 的关键工艺
- 薄膜沉积、掺杂等

> CPU 雕出"静态结构"，MEMS 雕出"可动结构" —— 这个类比**非常精准** 👍

### 加速度传感器
- 梳状结构（Comb Drive）✅
- 弹簧+质量块 ✅
- 差分电容检测位移 ✅

### 1. 梳齿结构的两种用途
梳状结构其实有两种角色：

| 角色 | 原理 |
|---|---|
| **感测轴**（面内x/y） | 梳齿**插入深度**变化 → 电容变化 |
| **驱动轴** | 反过来加电压驱动运动（陀螺仪里用） |
| **垂直轴 z** | 极板**间距**变化 → 电容变化（结构不同）|

### 2. 陀螺仪也是 MEMS
手机里除了加速度计，还有 **MEMS 陀螺仪**（Gyroscope）：
- 先用电**驱动**一个质量块持续振动（科里奥利力参考系）
- 再**检测**垂直方向因旋转产生的科里奥利力引起的位移
- 加速度计 + 陀螺仪合在一起叫 **IMU**（惯性测量单元）

### 3. MEMS 麦克风
- 振膜（Diaphragm）+ 背板（Backplate）构成平行板电容
- 声压 → 振膜位移 → 电容变化
- 现在几乎所有手机麦克风都是 MEMS 麦克风，已基本取代驻极体麦克风

### 4. 晶振

| 类型 | 物理结构 | 频率范围 |
|---|---|---|
| **AT切石英晶振** | 石英片剪切振动（厚度切变） | MHz级，最常见 |
| **音叉晶振** | 真的音叉形状，32768 Hz | 手表/RTC专用 |
| **MEMS振荡器** | 硅微谐振梁，纯半导体工艺 | 正在替代石英 |

> 32768 Hz = 2¹⁵ —— 正好是 **15次二分频**就得到 1Hz 秒信号，这是故意设计的！

### 5. DLP DMD
- 每个微镜约 **5~10 μm** 大小
- 通过静电力驱动偏转 **±12°**
- 1080p DMD 芯片上有 **207万个**独立微镜，同步翻转
- 这是目前**最复杂**的 MEMS 商业产品之一

```
MEMS 的本质 = 把"机械工程"缩小到微米尺度，
              用"半导体工艺"来"制造"，
              用"电信号"来"驱动/感知"
```

弹簧、质量块、阻尼、谐振——全是经典力学，只是尺度变了，量子效应还没显现，**经典物理完全够用**，这也是 MEMS 好分析的原因。
