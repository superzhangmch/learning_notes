# 地球历史时期动物与植物体型演化总览表

| 地质时期          | 距今（Ma）     | 动物体型特征                                   | 植物体型特征                          |
| ------------- | ---------- | ---------------------------------------- | ------------------------------- |
| **前寒武纪**（元古宙） | > 541      | 微米\~厘米级，多为软体、无器官结构，如海绵、疑似动物（Dickinsonia） | 微米\~厘米级，多为单细胞或简单多细胞藻类，无根茎叶，完全水生 |
| **寒武纪**       | 541 – 485  | 快速多样化，体型达几十厘米，如奇虾（Anomalocaris）          | 植物稀少，主要为绿藻等水生小型植物               |
| **奥陶纪**       | 485 – 444  | 海洋动物繁荣，部分腕足动物等体型达几十厘米                    | 最早的陆地植物出现（苔藓状），仍为几厘米以内          |
| **志留纪**       | 444 – 419  | 有颌鱼类出现，部分体长达1米以上                         | 最早的维管植物出现，如 Cooksonia（几厘米高）     |
| **泥盆纪**       | 419 – 359  | 鱼类大发展，板甲鱼类如邓氏鱼长达10米                      | 原始树木出现，如古裸蕨类高达10米，形成森林雏形        |
| **石炭纪**       | 359 – 299  | 昆虫和两栖动物巨型化，如巨蜻蜓翼展70 cm                   | 巨型植物繁盛，石松类高达30米，森林广布            |
| **二叠纪**       | 299 – 252  | 部分爬行动物崛起，大型掠食者体长几米                       | 干旱化后巨型蕨类减少，裸子植物兴起，树高10–30米      |
| **三叠纪**       | 252 – 201  | 恐龙初现，体长数米                                | 裸子植物继续主导，松柏、苏铁类为主，高达几十米         |
| **侏罗纪**       | 201 – 145  | 巨型蜥脚类恐龙出现，达30米、60吨                       | 与恐龙共生的裸子植物繁盛，如银杏、针叶树            |
| **白垩纪**       | 145 – 66   | 巨型恐龙继续繁荣，鸟类开始兴起                          | 被子植物（开花植物）兴起，体型多样，从草本到乔木皆有      |
| **古近纪**       | 66 – 23    | 恐龙灭绝，哺乳动物体型迅速增大，如巨犀达20吨                  | 被子植物广泛扩展，乔木与草本多样共存              |
| **新近纪**       | 23 – 2.6   | 大型哺乳动物丰富，如猛犸、地懒等，体长数米、高3–4米              | 现代植物体系基本确立，高大乔木与草原植物并存          |
| **更新世**       | 2.6 – 0.01 | 巨型哺乳动物仍多，但逐渐灭绝，人类影响增强                    | 农业植物开始出现，草类、谷类广布                |
| **全新世（现代）**   | 0.01 – 今   | 陆地最大动物为非洲象（4米），海中为蓝鲸（30米）                | 被子植物主导，最高为红杉（115米），多数植物体型适中或矮小化 |

总结趋势：

* **动物体型**：总体趋势为“从小 → 巨大化（中生代）→ 中小型化（现代）”
* **植物体型**：总体趋势为“从微小水生藻类 → 巨型森林（石炭纪）→ 多样化（现代）”
