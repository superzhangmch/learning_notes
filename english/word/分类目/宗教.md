天主教神职人员职位：
1. 平信徒（Laity）普通教徒，没有受圣职。
2. 圣职人员（Clergy）
- 第一级	Deacon	执事	协助神父、主教，能主持洗礼、婚礼，但不能主持弥撒。分常任执事和过渡执事（准备成为神父的）。
- 第二级	Priest	神父 / 司铎	主要负责主持弥撒、赦罪、施行圣礼，管理堂区（parish）。
- 第三级	Bishop	主教	管理整个教区（diocese），授予坚振圣事，按立神父和执事。
3. 高级主教（Higher Bishops） 
Archbishop	大主教	领导一个总教区（archdiocese），监督多个教区。有些特别重要的大主教被称为Metropolitan（都会大主教），他们还领导一个教省（province）。
4. 枢机（Cardinal）
Cardinal	枢机主教（红衣主教）	由教皇任命，属于教廷（罗马教会中枢），选举教皇，协助治理全球教会。	
5. 平信徒 (Laity) => 执事 (Deacon) => 神父 (Priest) => 主教 (Bishop) => 大主教 (Archbishop) => 枢机 (Cardinal) => 教皇 (Pope)
- 所有主教以上的等级（包括大主教、枢机、教皇）都被认为是主教身份的延伸（本质上是主教，只是责任和权力不同）。
