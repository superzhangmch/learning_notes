# 无线电发射与接收 QA笔记


## AM / FM 发射原理

**Q: AM调幅发射，是改变电压，保持电容/晶振不变吗？**

A: 基本正确。AM是改变载波的幅度，晶振或LC回路保持不变，所以频率稳定。实际电路中通过改变功放的电压/电流来实现幅度变化。
 
**Q: FM调频发射，是通过改变电容，从而改变频率吗？**

A: 完全正确。具体流程：
```
音频电压变化
    ↓
改变 变容二极管 的电容值
    ↓
改变LC振荡回路频率
    ↓
FM信号发射
```
变容二极管（Varactor）就是"压敏电容"，反向电压越高，电容越小，频率越高。

**Q: AM vs FM 对比？**

| | AM | FM |
|--|----|----|
| 改变的量 | 幅度（电压幅值） | 频率 |
| 不变的量 | 频率（晶振不动） | 幅度 |
| 实现方式 | 改变功放电压 | 改变变容二极管电容 |


## FM接收原理

**Q: FM 频率一直在变，接收端怎么共振？**

A: 接收端LC回路只需对准中心频率，不需要跟着跑。因为FM频率变化范围很小：
```
比如 98.0 MHz 电台：
最低：97.925 MHz
最高：98.075 MHz
变化范围只有 0.15 MHz
```
LC回路有一定带宽，中心频率附近的信号都能进来。

**Q: 共振不需要严格匹配频率，adjacent freqs都OK？**

A: 对！LC回路有"带宽"概念：
```
响应强度
    |
    |      ███
    |     █████
    |    ███████
    |________________→ 频率
           ↑
        中心频率
两边频率响应弱一些，但还是有！
```
FM电台带宽设计得刚好覆盖频率摆动范围。

**Q: FM频率在中心频率附近摆动，接收端怎么检测这个变化？**

A: 利用LC回路"离共振峰越近，输出电压越高"的特性。把LC回路的共振峰故意调偏：
```
FM信号的发射, 围绕在主频率 98.0 MHz 附近摆动
把接受端的 LC 回路峰值调到 98.2 MHz（故意偏高）

结果：
频率偏高 → 离峰值近 → 输出电压高
频率偏低 → 离峰值远 → 输出电压低
频率变化 → 自动变成电压变化！
```
峰值故意偏开，是为了让FM摆动范围落在斜坡上，而不是山顶上。山顶对称无法分辨，斜坡单调一目了然。

## LC电路与现代设备

**Q: 所有电磁波发送/接收，包括手机，都靠LC电路吗？**

A: 不完全是。

| 时代 | 器件 |
|------|------|
| 传统收音机 | LC回路 |
| 现代手机/WiFi | 晶体滤波器、SAW声表面波滤波器、锁相环PLL |

现代设备频率太高（4G/5G达到几GHz），LC电路无法制造得那么精确。但本质原理一样，都是找到对特定频率敏感的东西。

**Q: LC电路线圈圈数为什么影响频率？**

A: 
```
圈数越多
    ↓
每圈磁场叠加，L（电感）越大
    ↓
f = 1/(2π√LC) 中，LC变大
    ↓
频率越低
```
类比秋千：线圈越长（圈数越多）→ 秋千越长 → 摆得越慢 → 频率越低。
