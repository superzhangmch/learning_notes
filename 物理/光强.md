
# 点光源

**问：点光源，距离越远，光强衰减是平方反比？**

答：点光源是。光向四周辐射，球面积 $4\pi r^2$，能量守恒，故：

$$I \propto \frac{1}{r^2}$$

**问：Lux 测量值也是如此？**

答：是。Lux 即照度，$E = I/r^2$，距离倍增，照度减为四分之一。

**问：1mm 与 10mm 处，岂非相差百倍？**

答：不然。平方反比，需光源可视为点源方成立。极近处，LED 乃面光源，照度变化甚微，距离须远大于光源尺寸，方符合 $1/r^2$。

# 面光源 -- 朗伯体面光源

**问：显示器此类大面积光源，是否不同？**

答：迥异。显示器近似朗伯体面光源，一定距离内：

$$E = \pi L$$

$L$ 为亮度（cd/m²，即 nit），与距离无关。故屏前远近，照度几乎相同。

**问：何谓朗伯体光源？**

答：从任意角度观之，亮度 $L$（cd/m²）恒定不变者。其发光强度分布：

$$I(\theta) = I_0 \cos\theta$$

$\theta$ 为与法线之夹角。正面最强，侧面趋零。

---

**问：为何朗伯体光源偏偏是 cosθ，而非各向等强？**

答：有二解：

其一，几何：斜看面元，投影面积缩为 dA·cosθ，强度须同比缩小，亮度方能保持恒定。

最直观的解释：投影面积. 想象一个小发光面元 dA：

```
正面看：         斜着看：
  ___            ___
 |   |          /   /  ← 看起来更窄
 |dA |         / dA/
 |___|        /___/

```

正面看到面积 = dA, 斜着看到面积 = dA·cosθ, 发光面"看起来更小"(所以假设光源没厚度). 如果亮度（cd/m²）要保持不变, 则发光强度必须 ∝ cosθ. 这是从朗伯体定义倒推出的约束.

其二，热力学：两朗伯体平面相对，唯 cosθ 分布方能保证能量平衡，否则违反热力学第二定律。故 cosθ 非人为假设，乃自然界漫射之必然。

如果两个朗伯体平面相对：平面A  ←→  平面B, 能量传输必须平衡（热平衡） 只有 cosθ 分布才能保证能量守恒. 其他分布会导致：某方向能量堆积, 违反热力学第二law.

以上是假设朗伯体的定义(处处光强一样)倒退出的. 另外: Lambert 1760年测量，发现漫反射面（如白纸、石膏、哑光涂料）出射光强： $I(\theta) = I_0 \cos\theta$ , 就是 cos 的. 

为什么漫反射会是 cosθ? 微观图像：

```
入射光
↓
进入粗糙表面，经历无数次随机散射
↓
出射方向完全随机，均匀分布于上半球
```

上半球立体角均匀分布，投影到观测方向：

$$\text{有效通量} \propto \cos\theta$$

这个 $\cos\theta$ 是**纯几何结果**，与材料无关。所以漫反射的 $\cos\theta$：有微观解释（随机散射+几何投影).

发光体的 $\cos\theta$：大概也是同样道理吧?

**问：无限大朗伯体墙，墙前任意一点照度皆相同？**

答：然。墙上每一面元向上半球发光，积分之：

$$E = \int L cosθ \, d\Omega = \pi L$$

离墙近：可见面元少，但各面元 $\theta$ 小，cosθ 大，贡献强。

离墙远：可见面元多，但 θ 大，cosθ 小，贡献弱。

两者精确抵消，故照度与距离无关。

**问：贴墙测光，与远处测光，数值相同？**

答：正是。贴墙处，$\theta \approx 0°$，仅受正面直射光，$\cos\theta = 1$；远处，四面八方积分，二者数值相等。此即 $\cos\theta$ 分布之妙——能量均匀弥散于空间。

**问：灯墙照明，岂非最均匀？**

答：理论上是。房间内各点照度相同，无衰减。然无阴影，立体感差，故展览、艺术照明有时仍需点光源以造阴影。

**问：显示器在多大距离内，近似理想面光源？**

答：经验法则，距离小于显示器尺寸时，近似成立。以 27 寸屏（约 60cm）为例：

| 距离 | 近似程度 |
|------|---------|
| < 30cm | 接近理想 |
| 60cm | 轻微偏差（正常使用距离） |
| > 120cm | 明显偏离 |
| > 300cm | 接近点光源 |

日常使用距离恰在过渡区，故显示器以 cd/m²（nit）而非 lux 标定亮度，乃正确之选。
