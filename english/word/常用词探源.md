- just
  - 本义是: 正确的、正义的、符合规则的、恰当的. 等于: right, fair, proper, correct
    - a just decision
  - 后变副词, 表示的是: 恰当地, 正好/刚刚, 恰好
    - just enough 恰好符合要求
    - just right 刚刚好
    - I just arrived. 恰好在刚才这个时间点
    - That's just what I wanted. 正好是
    - 仅仅的意思, 仍然来自“正好”: 演化路径: 正好这么多 → 不多不少 → 只有这么多 → 仅仅
      - just one minute / I just have $10. 
  - 后来: 精确副词 → 语气缓冲词
    - I just wanted to ask…; 我只是想问一下…
    - 字面上把行为范围说小了 → 语气自然就轻了

- even
  - 核心原义只有三个字：平、齐、等
    - an even surface
    - even numbers; 偶数（能被2整除 → 可均分 → 平分）
  - 甚至:
    - 关键语义跳跃：从“相等” → “也算在内”. 比如: X is true. Y is even true. Y 和前面那些一样成立
    - 发展出“甚至”义:
      - Even a child can do it. 成人可以 → 青少年可以 → 小孩也一样可以 → 小孩 even included → 连小孩都能. 平等纳入集合 → 极端点也被纳入
      - 强调: He didn’t even call.
  - 为什么“even”能表示“更强强调”:
    - 把一个“更极端的点”拉到和普通情况同一水平线上. even = 把例外拉平

- yet
  - 本义: 到现在为止仍然存在. 它的核心语义一直是：时间延续 + 状态未结束
  - It is yet warm. 仍然
  - He hasn’t arrived yet. 因为“时间延续”，就自然产生：not yet = 到现在还没有
  - 从“仍然 + 继续” → “再进一步”(仍然有“仍然”, “延续”的意思): yet another 又一个 / 甚至再一个; yet more problems 更多问题
  - 转折: 情况本应结束 → yet → 居然仍然存在 → 形成转折感
    - He is poor, yet happy. 却

- ever
  - 本义: always / at any time,
    - 在任何时间点上, 贯穿时间轴, 不设时间限制
  - 保留原始义 一直: for ever, ever living, ever after
  - 从“始终” → “任意时间点”, 曾经: Have you ever seen snow? == have you at any past time seen snow?
  - 在任何可能时间里: if ever, hardly ever, never ever
  - 强调语气副词（现代高频）:
    - Who ever told you that? 这到底是谁告诉你的？ / What ever are you doing?你究竟在干什么？
    - Can a machine ever love you? “究竟 / 到底 / 有没有可能”
  - 相关词或短语:
    - forever = for + ever = 在所有时间中
    - however / whatever / whoever / whenever: 词源逻辑：在任何情况下, 在任何人, 在任何时间
    - ever since = 从那之后一直; ever more / ever greater = 持续增强
    - never = not ever

- the, that, there, than, then 同源 
  - do you see the book, 其实就是 do you see that book
  - than 也来自 that.
    - He is taller than me. 深层逻辑其实是：他更高 —— 相对于“那个人”. 也就是：taller — that one.
      - than 本质上是：把比较对象“指出来”
  - than 和 then 原来是一个词.
    - than: 专门用于比较; then: 专门用于时间

- so = in this way = 像这样
  - 本义: “这样地 / 按这种方式”, in this way / like this

- such
  - 词源上 such=so+like. 同样构词法的词: childlike, 像孩子的
    - "such" 来自古英语 "swylc/swilc",是由 "swa"(so)+ "lic"(like/body/form)组成的

- as
  - 本义: "同样地"（in the same manner/degree）
  - 古英语：ealswā / alswā, 结构 = all + so
    - 意思 = exactly so / just like that / in that way
    - 中文可理解为：“正如那样 / 就那样地”
  - 用法
    - as tall as him 像……一样
    - Do as I say. 按……方式
    - He works as a teacher. 以……的身份
    - As I was walking, it started to rain. 当……的时候.  在我处于走路这个状态中时 —— 雨发生了, 乃时间背景对齐
    - As it was late, we left. 原因. 在“已经很晚”的这个状态框架下 —— 我们离开, 乃条件背景对齐
  - as 来自 all + so，所以：as = so = that way
    - 很多古结构现在还保留：
      - as…as
      - so…as（古/正式）
      - such as
  - as 是“方式 / 状态 / 对等关系”的指示词, 可以把它看成一个“状态对齐词”
| 用法           | 深层含义 |
| ------------ | ---- |
| as tall as   | 状态一样 |
| do as        | 方式一样 |
| as a teacher | 身份一样 |
| as I said    | 情况一样 |
| as it rained | 条件一样 |

- also
  - also = all + so, 所以意思是: 意思是：just so / in the same way / equally, 正是如此 / 同样地
    - also 和 as 非常同源, 都算是 all + so

- to, too
  - to 和 too 是同源词
  - 本义: 朝向, 向前延伸直到某点
    - 延伸到目标 → to
    - 延伸过界 → too
  - to
    - go to the city 空间移动
    - give it to him 抽象方向
    - from A to B
    - to learn 朝向学习这个动作; 方向 → 目标 → 目的; 这就是 不定式 to 的来源
      - 古英语不定式, 本来是: `to + 动名词（verbal noun）` 意思是 `toward doing sth`, 后来简化为 `to do sth` 
    - to 始终只有一个核心概念：朝向一个目标
      - 同词根词 toward
  - too: 延伸过头 / 超出限度
    - too big  = 大过头
    - too much = 多过头
    - 也: 已有集合+再延伸一个成员 -> too; 词源逻辑：把范围再延伸一格

- can, may, shall, must, will
  - shall
    - You shall go. 原始感觉不是: 将来, 而是：你有义务去
    - 为什么会变成将来时标记: 义务 → 必然发生 → 将来

| 词     | 原始词根              | 古英语形式  | 最早含义     | 语义演化路径                |
| ----- | ----------------- | ------ | -------- | --------------------- |
| can   | PIE *ǵnō- = 知道    | cunnan | know how to, 知道如何     | 知道 → 会 → 有能力 → 能      |
| may   | PIE *magh- = 有力量  | magan  | 有力量      | 有力量 / 有能力 → 能 → 被允许 → 可以/可能 |
| shall | PIE *skel- = 欠债   | sceal  | 欠债, 负有义务      | 欠债 → 应当 → 必须 → 将要     |
| must  | Proto-Germ *motan | mōtan  | 被迫 / 被允许 | 被允许 → 被迫 → 必须         |
| will | PIE *wel- = 愿望 | willan | 意愿, 想要 | 想要 → 意愿 → 打算 → 将来 |

- who, what, who, which, when, why, how
  - 同源. how 其实也算是 wh 开头的.

- either, neither, whether
  - neither = not + either, whether 可理解为 wh询问 + either, 即二者中选哪一个

- with, without
  - with 本义: against / toward / opposite, 对着、朝向、对抗
    - 古义: fight with someone, 和某人对打
    - 保留古义的:
      - withstand = with + stand = 站着对抗 = 抵抗
      - withdraw = with + draw = 向反方向拉 = 撤回
      - withhold = 扣住不给 = 保留
  - 后来变成“和……一起”: 对着 → 面对 → 接触 → 一起 → 伴随
    - I go with you. 和……一起
    - a man with a hat 带着
    - cut with a knife 用
    - trembling with fear 因为（情绪/状态伴随）
    - angry with him 处理对象
    - argue with someone = 和某人争论 / 跟某人吵
    - fight with someone: 可以两种解释 和某人并肩作战/和某人打架（互相对打）

- by
  - 本义: around / near / on both sides / nearby; 在周围、在旁边、靠近
    - 最初是一个空间位置词：在……旁边, 靠近, 邻近: sit by the fire
  - 从“在旁边” → “沿着” → “经过”: pass by 从旁边经过
  - 从“经过某路径” → “通过某方式”: by train, by hand
  - 从“通过某方式” → “由某人完成”: written by him
  - increase by 10% 变化是“沿着数值轴移动”

- of (和 off 同源)
  - 本义: away / off / from, 离开、从……分离 
  - 从“离开” → “来源于”: a man of Rome
  - 从“来源于” → “属于”: the son of the king
  - 从“来源于整体” → “是其中一部分”: a piece of cake
  - 材料 / 构成: a cup of gold
  - 抽象连接词： the problem of time 其实是：关于 / 源于 / 关联于
  - 它从一个空间分离词，逐渐抽象化为：关系标记符, 类似于：中文的“的”, 但词源上它其实不是“的”，而是：从……出来的

- for
  - 本义: 在前面 / 朝向前方 / 面对
    - for 和 forward、fore、before 是亲戚
      - fore 同词根词: forearm, forehead, forecast
    - 朝向:
      - go for him → 朝他那边去
      - stand for → 站在前面
  - 方向 → 目标: 朝向一个目的
    - for victory → 为了胜利
    - for freedom → 为了自由
  - 朝向 → 受益对象
    - This is for you
    - I made it for her
  - 朝向 → 交换 / 代替; “朝向某个对象” → “用来换取”
    - $10 for the book
    - trade A for B
  - 朝向 → 原因: 动作朝向某种结果 → 因为
    - punished for cheating
    - famous for music

- rather
  - 原本是：某一表示 fast 的比较级 —— 意思是 “更快地”
  - 更快 → 更早: “更快做某事” → “更早做某事”
    - I would rather go ≈ I would sooner go
      - rather: “更倾向”, 宁可. sooner(更早) 也可以表示“宁愿”, 看来异曲同工
  - 更早 → 更倾向: “更早去做 A 而不是 B”, 就变成：你更愿意 A, 于是语义变为：
    - I’d rather stay home = I prefer to stay home
  - rather than 为什么表示“而不是”:
    - rather = 更倾向; than = 比
    - rather than complain, act = 更倾向行动，而不是抱怨
  - 相当、颇: 更 → 稍微更 → 有点偏向 → 相当
    - rather good, rather strange, rather difficult

- very
  - 本义: true / real / genuine, 真的. 和 verify 同源
    - 体现本义的用法:
      - the very God = 真神 / 真正的神
      - the very same man = 正是那个人
      - the very idea = 这个想法本身
      - at the very moment = 就在那个时刻
      - the very person I wanted = 正是我想要的那个人
      - 这里的 very = exact / real / true

- per
  - 本义: 穿过, 通过, 越过, 贯穿
  - 每: through each unit → per unit → 每单位; 穿过每一个单位
    - miles per hour, price per kg, income per year
  - 根据: per = according to, 取“通过” 之意. per 只是指出执行动作时的依据来源
    - Install the software per instructions. 字面意思: 通过说明作为依据执行

- but
  - 本义: outside, without, except = 在外面、没有、除……之外.
    - 最开始写作 būtan / butan: be- = by / about（在旁边）; ūtan = outside（外面）
    - 早期 but 的意思接近今天的：except, besides, apart from
      - 今天仍用:
        - No one but me knows / All but one survived / Nothing but trouble / He is good but poor / It’s nothing but a joke
        - 几乎 = 除了那一点点之外都..: all but finished
        - But for your help, I’d fail. 要不是你的帮助…
  - 为啥表示但是: 除外 → 排除前面说的情况 → 引入不同情况 → 转折 → 但是
    - 前面说一个情况, but = 把它“放到外面” → 引入另一种不同情况 → 转折语气产生

- because
  - because = be + cause = by cause that = 因为……

- usual/ **used** to do
  - 同源（都来自 use / usus）
  - 含义相关（都和“习惯 / 反复行为”有关）
  - 语义演化逻辑是同一条主线
  - 从：反复使用形成的习惯, 语义自然扩展成：惯常的 → 通常的

- due, due to/owing to
  - due 本义: “被欠的、应当给出的” → owed
    - due / debt / duty / debit 同源
    - due to = owing to = because of; due 与 owed 意思一样, 所以 due to, owing to 意思一样
  - 例句
    - The payment is due tomorrow. 款项明天到期（该付了）
    - respect is due **应得的**尊重
    - with due care 以**应有的**谨慎
    - In due course the baby was born. 婴儿如期降生了

- over
  - 本义: 在某物之上 / 覆盖在上方
    - a roof over the house 房子上面的屋顶
    - a cloth over the table 桌子上覆盖着布
  - 上方 → 跨越 jump over the wall / fly over the city
  - 跨越 → 超过 over 100 people / over budget / over the limit
  - 覆盖 → 控制 → 支配 rule over a country / authority over others / power over
  - 覆盖 → 遍及 → 全部 all over the world / all over the place
  - 越过 → 结束 the game is over / it's over now
  - 翻过去 → 重来 start over / do it over
  - 一些例子
    - pull over: 在路中间行驶 → 把车“拉”到路边; 从当前路线“越过去” → 到边上去
    - go over: go over 从上面走过去 → 把整个表面走一遍, 于是语义隐喻变成：把内容从头到尾过一遍
    - look over 眼睛从上方扫过去, 视线 ——→→→ 扫过物体, 所以变成：快速浏览 / 大致检查

- each, every
  - each: 古英语 ǣlc 由 ā（ever，永远）+ ge-（前缀）+ līc（like，相似）组合而成, 字面意思："ever alike"（永远相似的）→ 每一个都一样
    - “ever-like” → “每一个个体都一样地被对待”
  - every: 古英语是 ǣfre + ǣlc 组合词, ǣfre = ever（总是）, ǣlc = each（每个）
    - ever-each = 始终每个 → 所有个体全部覆盖
